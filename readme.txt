WPC AJAX Search is an interaction search popup for WooCommerce.

== Description ==

A brand-new plugin for solving the puzzle of matching the people in need with the products indeed. **WPC AJAX Search** enables visitors on your site to enter the search popup from anywhere the search button is placed and get instant results with a quick preview of product details.

This is one of the most convenient search technologies that is an absolutely perfect solution for store owners to make sure people find what they need, get great recommendations of what they are possibly interested in, and effortlessly add them to the cart. Especially, for wholesalers or giant businesses, the searching experience is crucial to the running of the business. This, thus, enhances the user experience and converts more visitors into customers effectively.

= Live demo =

Visit our [live demo](https://demo.wpclever.net/wpcas/ "live demo") here to see how this plugin works.

= Key Features =

- Live search, instant results of products
- Set up suggestions for special keywords
- Add search to different menus
- Show/ hide the overlay effect
- Click outside to hide the popup
- Add products to the Wishlist & Compare right from the search popup
- Automatically show or hide the search popup when search input is possible
- Choose a heading for the search popup
- Choose a position for the search popup on your site
- Enable/ disable the sliding effect of the popup
- Show/hide the Close button at the top corner of the popup
- Link to individual products in the same or a new tab
- Compatible with most common WordPress themes and plugins
- Open individual products in a Smart Quick View popup
- Set up keywords for one-sale, recent, featured, and popular products
- Configure the search settings: by categories or SKU, search limit
- Compatible with WPML for building multilingual sites
- RTL support for better displaying right-to-left languages
- More features to be updated

**WPC AJAX Search** can work with almost any theme and system. Lightning-fast speed results & compatibility with WPC Smart Quick View allows buyers to preview the products or even add them to the cart right from within the popup (with the Premium version of Smart Quick View). To get a nice effect for notifications when adding items to the cart, you can also install the **WPC Added to Cart Notifications** plugins.

== Installation ==

1. Please make sure that you installed WooCommerce
2. Go to plugins in your dashboard and select "Add New"
3. Search for "WPC AJAX Search", Install & Activate it
4. Go to WP-admin > WPClever > AJAX Search to configure
